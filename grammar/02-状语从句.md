### 什么是状语
- 描述状态, 时间, 地点, 原因 ...

- 时间状语 => when, while
- 地点状语 => where
- 原因状语 => because, for , as , due to
- 目的状语 => in order, so that
- 条件状语 => if, unless, as/so long as, lest, in case
- 结果状语 => so, so that
- 让步状语 => as if, although, though
- 方式状语 => by way of, by means of
- 比较状语 => than as as, not as as, no sooner than
